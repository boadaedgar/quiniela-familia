📦 ¿Qué es npm?

npm es el sistema que descarga e instala herramientas de programación con un solo comando — como un delivery de software que llega en segundos.

El nombre significa Node Package Manager — el administrador de paquetes de Node.js.

Metáfora: Pensalo como el mercado de apps de tu celular, pero para herramientas de código. En vez de buscar, descargar e instalar manualmente, escribís el nombre de lo que querés y npm lo hace todo solo — en segundos, con la versión correcta.

⬇️ ¿Cómo se instala?

1
npm viene incluido con Node.js ✓ gratis — no se instala por separado
2
Vas a nodejs.org y descargás el instalador (como cualquier programa normal)
3
Doble clic en el instalador → Siguiente → Siguiente → Finalizar
4
Abrís la terminal y escribís npm --version — si aparece un número, ya está listo ✅
Lección · Slide 4 de 5

🤔 ¿Por qué lo necesitamos?

Sin npm vs con npm ❌ Sin npm — para instalar una herramienta tenés que: buscarla en internet · descargar los archivos · configurar todo a mano · actualizar cada parte por separado · rezar para que funcione

✅ Con npm — escribís un comando y listo:
npm install slidev

npm descarga la herramienta, todas sus dependencias, la versión correcta, y deja todo configurado — en segundos.
