🔌 ¿Qué es una API?

Una API es un mensajero digital que conecta dos programas: lleva el pedido de uno y trae la respuesta del otro.

Metáfora: Estás en un restaurante. Vos sos el cliente (tu app), la cocina tiene la comida (los datos), y el mozo que toma tu pedido y te trae el plato es la API. Nunca entrás a la cocina — el mozo hace todo el trabajo del medio.
Lección · Slide 3 de 5

⚙️ ¿Cómo funciona?

  • Tu app hace un pedido: "Dame el clima de Buenos Aires hoy"
  • La API lleva ese pedido al sistema que tiene los datos del clima
  • El sistema busca la información y prepara la respuesta
  • La API trae esa respuesta de vuelta a tu app — y la ves en pantalla

🗺️ Un ejemplo que ya usás

Ejemplo real — Google Maps Cuando abrís Google Maps en tu celular, la app usa una API para pedirle al servidor de Google los mapas actualizados, el tráfico en tiempo real y las rutas más rápidas.

Sin la API, Maps no sabría cómo pedirle esa información a Google, ni cómo recibirla.
Con la API, todo pasa en menos de un segundo — sin que vos tengas que hacer nada.
