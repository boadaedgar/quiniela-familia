🟨 ¿Qué es JavaScript?

JavaScript es el lenguaje que hace que las páginas web sean interactivas — botones que responden, menús que se abren, formularios que validan. Toda página web usa tres ingredientes: HTML estructura · CSS diseño · JavaScript comportamiento

Metáfora: Si una página web fuera una casa — el HTML son las paredes y el techo, el CSS es la pintura y la decoración, y el JavaScript es la electricidad: lo que hace que las luces prendan, las puertas se abran y la heladera funcione.

⚙️ ¿Y Node.js qué tiene que ver?

  • JavaScript solo funciona dentro de un navegador — Chrome, Firefox, Edge
  • Node.js es un programa que permite correr JavaScript fuera del navegador, directo en tu PC
  • Muchas herramientas de desarrollo están escritas en JavaScript — necesitan Node.js para funcionar
  • Al instalar Node.js también instalás npm — el sistema que descarga esas herramientas

🛠️ Por qué Slidev lo necesita y Reveal.js no

Lo que pasó en nuestro proyecto Cuando exploramos Slidev, el sistema pedía correr:
npm create slidev my-slides

Eso requería Node.js instalado en tu PC — porque Slidev es una herramienta escrita en JavaScript que corre fuera del navegador.

Reveal.js con CDN no lo necesita — el archivo HTML se abre directo con doble clic, el navegador hace todo.

Por eso elegimos Reveal.js para empezar: cero instalación.
