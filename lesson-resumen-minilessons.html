🗂️ Todo junto en el proyecto quiniela

Cómo se usan los 8 conceptos en un solo proyecto
HTML + CSS arman la página que ve la familia — tabla de resultados, colores, botones
API trae los resultados de los partidos desde internet en tiempo real
JS + Node.js calcula los puntos de cada apostador y actualiza la pantalla
npm instala las herramientas de desarrollo que se necesiten
Git guarda cada versión del proyecto — si algo se rompe, se vuelve atrás
Markdown documenta cómo funciona todo en archivos .md legibles
Skills + IA generan lecciones, código y assets — el humano dirige, la IA construye
