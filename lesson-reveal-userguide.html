One command. A full interactive guide.

/reveal-userguide <topic> generates a complete, multi-section HTML presentation with horizontal sections, vertical sub-slides, tool tables, workflow steps, and a quick reference card — all in a single file you can open anywhere.

Metaphor: It's like a printing press for slide decks. You name the topic, and a fully designed, structured guide comes out the other side — ready to hand to anyone.
Lesson · Slide 3 of 5

How it works

  • Type /reveal-userguide <topic> — the skill takes it from there
  • It builds horizontal sections (big topics) with vertical sub-slides (details) inside each
  • Covers: what the tool is, how to install it, key commands, step-by-step workflows, and a quick reference card
  • Output is a single self-contained HTML file — no internet needed to view it, no app to install
  • Double-click the file → opens in any browser; use ← → ↑ ↓ arrow keys to navigate
