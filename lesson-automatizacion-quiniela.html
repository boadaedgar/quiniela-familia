La Gran Idea

La quiniela ahora le pregunta a un servicio externo cuáles son los marcadores del Mundial, en lugar de esperar que alguien los escriba a mano.

Metáfora: Es como el pizarrón de llegadas en un aeropuerto — vos no actualizás los vuelos uno por uno. El pizarrón consulta el sistema del aeropuerto y se actualiza solo. Tu quiniela hace lo mismo con los resultados del Mundial.
Lección · Slide 3 de 5

¿Cómo funciona?

  • La página carga → automáticamente consulta worldcup26.ir, una API gratuita del Mundial 2026
  • La API responde con todos los partidos: equipos, marcador, y si el partido terminó o no
  • El sitio traduce esa información a su formato interno (m01, m02… m24) usando una tabla de equivalencias
  • La tabla de posiciones, los puntos, y los partidos se recalculan y se muestran solos — sin recargar
API = una puerta de acceso a datos de otro sistema. Le pedís algo, te responde con información.
