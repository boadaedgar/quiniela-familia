Toda página web que abrís tiene exactamente dos ingredientes

No importa si es Google, Instagram, un banco o un blog — toda página web del mundo está construida con las mismas dos cosas.

Una define qué hay en la página. La otra define cómo se ve. Las dos juntas hacen lo que ves en tu pantalla.

🏗️ El dúo que construye toda página web

HTML pone el contenido — textos, imágenes, botones, títulos.
CSS lo viste — colores, tamaños, posiciones, fuentes.

Sin HTML no hay nada. Sin CSS, todo es texto negro sobre fondo blanco.

Metáfora: Construir una casa. El HTML son los ladrillos, las paredes y las ventanas — la estructura. El CSS es la pintura, los muebles y la decoración — lo que hace que se vea bien. Podés tener paredes sin pintura, pero no pintura sin paredes.

⚙️ Cada uno hace su trabajo

HTML — el contenido

Define qué existe en la página <h1>Quiniela FIFA</h1> <p>Argentina 2 - Francia 0</p> <button>Actualizar</button>
→ Aparece texto, párrafo y botón.
Sin color, sin estilo.

CSS — el diseño

Define cómo se ve cada elemento h1 { color: orange; } p { font-size: 18px; } button { background: blue; border-radius: 8px; }
→ El título es naranja, el texto grande,
el botón azul y redondeado.

🖥️ Los dos en nuestro proyecto

Las lecciones que generamos juntos Cada archivo lesson-*.html que creamos en este proyecto usa HTML y CSS al mismo tiempo:

HTML define las secciones de cada slide, los títulos, el texto de las preguntas y las opciones del quiz.

CSS define los colores naranja y azul, el tamaño de la letra, los bordes redondeados de los cajones y el fondo oscuro.

Los dos viven en el mismo archivo — el CSS va dentro de la etiqueta <style> al principio del HTML.
